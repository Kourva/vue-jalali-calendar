# Vue/Nuxt Jalali Calendar
A tiny but powerful Jalali calendar for Vue/Nuxt — written in pure TypeScript, zero‑dependency, and shipped as a drop‑in component folder (no package install needed). Features dual Jalali/Gregorian view, 100+ stylish themes (yes, even Spotify &amp; SoundCloud), and a wide set of customization options for full control.
